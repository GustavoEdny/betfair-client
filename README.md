## Betfair Client
#### A simple and lightweight Betfair API wrapper

##### Installing

Using YARN

```
yarn add @gustavoedny/betfair-client
```

Using NPM

```
npm install --save @gustavoedny/betfair-client
```

It currently supports:

| API              |-----|
|------------------|:---:|
| Authentication   |  ✔️  |
| Betting          |  ✔️  |
| Account          |  ❌ |
